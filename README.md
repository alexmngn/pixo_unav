# pixo_unav.js
A simple navigation menu with coloring underline on the position of the current active item


## Compatibility

* jQuery 1.7.1+
* Internet Explorer 7+
* Google Chrome
* Mozilla Firefox
* Safari
* Opera

## Usage

```javascript
$('ul#menu').pixo_unav();
```
